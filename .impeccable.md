# Impeccable Design Context

## Design Context

### Users
Homelab enthusiasts and solo developers who live in the terminal all day. Managing personal servers, home infrastructure, jump hosts, database tunnels. Deeply technical, comfortable with SSH internals, and they appreciate tools that are smart and well-crafted.

### Brand Personality
**Three words**: nerdy, opinionated, delightful.
The physical object: a clean lab notebook — structured, methodical, but personal. It has a system, it has annotations, it has its own logic.

### Aesthetic Direction
- **Theme**: Dark. This tool lives in the terminal; the landing page should feel at home there.
- **Palette**: Deep navy background (not pure dark, tinted toward teal), warm amber accent. Deliberately avoids the default cyan-on-dark aesthetic.
- **Typography**: Chakra Petch for display (technical, structured, slightly unusual), Atkinson Hyperlegible for body (very readable, slightly quirky), Fira Code for code.
- **Layout feel**: Graph-paper grid texture, numbered sections, ruled separators — like annotated lab notebook pages.
- **Anti-references**: No generic developer landing pages with green-on-black terminals. No purple-gradient "AI product" look. No glassmorphism. No giant hero icons above every heading.

### Design Principles
1. **The Icelandic name is a feature**: Render "Tengingarstjóri" large and proud with phonetic pronunciation. Own the unusual name.
2. **Methodical over flashy**: Structure and clarity over decorative complexity. Every element earns its place.
3. **Terminal-literate but not terminal-clichéd**: Reference the terminal context tastefully — no fake green-on-black windows.
4. **Delight lives in the details**: Small touches of personality (annotation-style command descriptions, lab notebook numbering) over gratuitous effects.
5. **Show the tool working**: Real command output, real table data — demonstrate rather than describe.
